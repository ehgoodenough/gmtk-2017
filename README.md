# gmtk-2017 #

A game for the Game Maker's Toolkit Jam. :]

### Getting Started ###

Have node v6+ and npm v3+.

```
node --version
npm --version
```

Install all the dependencies.

```
npm install
```

Build the game!

```
npm start
```

To learn more about our build system, check out [codeglue](http://github.com/ehgoodenough/codeglue).
